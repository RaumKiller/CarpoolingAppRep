# Documentación del Proyecto: Carpooling App

Esta carpeta (`docs/`) es el repositorio central de toda la documentación técnica y de negocio del proyecto. 

Como tu Ingeniero de Sistemas y Desarrollo de Software asignado, utilizaré los archivos colocados aquí para entender las reglas de negocio, la arquitectura y los esquemas de base de datos que definas.

### Archivos Sugeridos (Puedes crearlos aquí más adelante):
- `database_schema.md`: Para el modelo relacional de Oracle XE 21C.
- `api_endpoints.md`: Diseño de la API REST que expondremos con Node.js/Next.js.
- `architecture.md`: Flujos de trabajo, contenedores Docker y cómo interaccionan el frontend y el backend.
